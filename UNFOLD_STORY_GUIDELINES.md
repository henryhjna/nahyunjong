# 언폴드 스토리 작성 가이드라인

## 📖 핵심 원칙

### 1. 소설처럼 읽히는 자연스러운 스토리
- **감정선과 캐릭터의 내면** 묘사
- 장면 묘사를 생생하게 (시간, 장소, 분위기, 감각)
- 대화를 통한 자연스러운 정보 전달
- 긴장감과 이완의 리듬

### 2. Markdown 문법 사용 금지
❌ **절대 사용 금지:**
- `**볼드**` → 그냥 강조하고 싶으면 문맥으로 자연스럽게
- `## 제목` → 이모티콘이나 자연스러운 문장으로
- `- 리스트` → 자연스러운 문장이나 줄바꿈으로
- `[링크](url)` → URL은 그냥 텍스트로

⚠️ **예외: learningPoint는 구조화된 설명이 필요하므로 제한적 사용 허용**
- 제목에만 `**볼드**` 허용
- 리스트 `•` 사용 가능

### 3. 숫자 표기 규칙
✅ **올바른 표기:**
- 100,000,000원 (콤마 사용)
- 1억원 (한글 단위도 OK)
- 2,450,000원 또는 245만원

❌ **잘못된 표기:**
- 100000000원 (콤마 없음)
- 1억 (단위 누락)

### 4. 이모티콘 사용 원칙

#### 📅 날짜/시간 표기 (필수)
```
📅 2022년 3월 2일, 목요일 오전 9시
```

#### 💬 대화/통신
- 💬 카톡, 메시지
- 📱 전화, 휴대폰 알림
- 📞 전화 통화 중

#### 💰 금융/거래
- 💰 금액, 잔액, 매출
- 💸 지출, 이체
- 💳 카드 결제
- 💵 현금

#### 📦 비즈니스 활동
- 📦 배송, 제품
- 📋 계약서, 서류
- 🛒 주문, 쇼핑
- 🏢 사무실, 부동산

#### 🎯 감정/상태
- 🔑 중요한 순간
- ⏰ 알람, 시간
- ✨ 희망, 설렘
- 🌃 밤, 도시 풍경
- 🤝 악수, 합의
- 🌐 웹사이트

#### ⚠️ 주의사항
- 이모티콘은 **정보 전달용**으로만 사용
- 문장 끝에 이모티콘만 덩그러니 두지 않기
- 한 씬에 너무 많이 사용하지 않기 (3-5개 정도)

## 📝 스토리 구조

### Intro (도입부)
- 200-300자 정도
- 시간적 배경, 인물 소개, 상황 설정
- 독자의 호기심을 자극
- 주인공의 내면 묘사

**예시:**
```
서울 강남, 이른 봄의 햇살이 따스한 3월 아침.

박유진은 거울 앞에 섰다. 서른두 살. 화장품 회사에서 5년간 연구원으로 일하며 쌓은 노하우와 저축 1억원이 전부였다. 손에는 기술보증기금의 대출 승인 서류가 들려 있었다. 2억원이라는 숫자가 눈에 들어왔다.

"정말 시작하는 거야? 안정적인 직장을 버리고?"

하지만 마음 한구석에선 이미 결정이 나 있었다. 지난밤 실험실에서 완성한 그 포뮬러, 민감성 피부를 위한 완벽한 세럼이 머릿속을 맴돌았다.
```

### Scene (장면)
각 장면은 **하나의 회계 거래**에 대응됩니다.

#### 필수 구성 요소:
1. **날짜/시간 표기** (📅)
2. **장소와 상황 묘사**
3. **인물의 행동과 대화**
4. **감정과 내면 묘사**
5. **회계적 사실을 자연스럽게 녹여내기**

#### 작성 팁:
- **구체적인 감각 묘사:** 시각, 청각, 촉각, 후각
- **대화를 통한 정보 전달:** 딱딱한 설명 대신 자연스러운 대화로
- **숫자는 맥락 속에 자연스럽게:** "통장에 100,000,000원이 찍혔다"
- **주인공의 감정 변화 추적:** 두려움 → 결단 → 희망 등

**예시:**
```
📅 2022년 3월 2일, 목요일 오전 9시

은행 창구 앞. 유진은 새로 만든 법인 도장을 손에 쥐고 있었다. 아직 잉크 냄새가 났다.

"안녕하세요, 법인 통장 개설하러 왔습니다."

목소리가 약간 떨렸다. 직원이 친절하게 미소 지으며 서류를 확인했다.

"주식회사 언폴드, 박유진 대표님이시군요. 자본금 납입은 얼마로 하실 건가요?"

"1억원입니다."

5년이었다. 월급의 반을 저축하고, 야근 수당을 모으고, 보너스를 한 푼도 쓰지 않고 모은 돈. 부모님께 손 벌리기 싫어서, 투자자를 찾는 것도 부담스러워서, 오로지 혼자 힘으로 모은 돈.

통장에 100,000,000원이라는 숫자가 찍히는 순간, 눈물이 핑 돌았다.

"축하드립니다, 대표님. 사업 번창하시길 바랍니다."

은행을 나서는 길, 봄바람이 불었다. 유진은 스마트폰으로 통장 잔액을 다시 한번 확인했다.

💰 잔액: 100,000,000원

"정말 시작이구나..."
```

### Learning Point (학습 포인트)
회계 개념을 쉽게 설명하는 섹션입니다.

#### 구조:
1. **제목:** 이모티콘 + 개념 이름
2. **한 줄 요약:** 핵심을 단순하게
3. **상세 설명:** 이모티콘 소제목으로 구분
4. **실무 팁:** 왜 중요한지, 어떻게 처리하는지

**예시:**
```
💡 자본금(Capital Stock)이란?

주주가 회사에 투자한 돈입니다. 유진이 개인 돈 1억원을 회사에 투자했으니, 이제 이 돈은 회사의 돈이 됩니다.

✨ 특징:
• 갚을 필요가 없어요 (빌린 게 아니라 투자받은 거니까!)
• 회사가 망하지 않는 한 돌려주지 않아도 돼요
• 자본(Equity) = 회사의 순수한 자기 자산

📊 회계 처리:
• 차변(왼쪽): 현금 100,000,000원 증가
• 대변(오른쪽): 자본금 100,000,000원 증가
```

## 🎬 장면 작성 체크리스트

### Before Writing
- [ ] 이 장면에서 일어나는 회계 거래가 무엇인지 명확히 파악
- [ ] 주인공의 감정 상태 설정 (불안? 흥분? 고민?)
- [ ] 장소와 시간대 구체화

### While Writing
- [ ] 날짜/시간 표기 (📅) 있는가?
- [ ] 구체적인 장소 묘사가 있는가?
- [ ] 대화가 자연스러운가?
- [ ] 숫자에 콤마를 찍었는가?
- [ ] Markdown 문법을 사용하지 않았는가?
- [ ] 회계 정보가 스토리에 자연스럽게 녹아들었는가?

### After Writing
- [ ] 소리 내어 읽었을 때 자연스러운가?
- [ ] 감정 흐름이 매끄러운가?
- [ ] 이전 장면과의 연결이 자연스러운가?
- [ ] 독자가 공감할 수 있는가?

## ❌ 나쁜 예시 vs ✅ 좋은 예시

### 나쁜 예시:
```
**3월 2일 자본금 납입**

박유진은 은행에 갔다.
- 자본금: 100000000원
- 계좌 개설
- **중요**: 자본금은 자기자본이다

회계처리:
(차) 현금 100,000,000 / (대) 자본금 100,000,000
```

**문제점:**
- Markdown 문법 남용 (`**`, `-`)
- 숫자 콤마 누락
- 감정 없음, 건조함
- 회계 용어 나열

### 좋은 예시:
```
📅 2022년 3월 2일, 목요일 오전 9시

은행 창구 앞. 유진은 새로 만든 법인 도장을 손에 쥐고 있었다. 아직 잉크 냄새가 났다.

"안녕하세요, 법인 통장 개설하러 왔습니다."

목소리가 약간 떨렸다. 직원이 친절하게 미소 지으며 서류를 확인했다.

"주식회사 언폴드, 박유진 대표님이시군요. 자본금 납입은 얼마로 하실 건가요?"

"1억원입니다."

5년이었다. 월급의 반을 저축하고, 야근 수당을 모으고, 보너스를 한 푼도 쓰지 않고 모은 돈. 부모님께 손 벌리기 싫어서, 투자자를 찾는 것도 부담스러워서, 오로지 혼자 힘으로 모은 돈.

통장에 100,000,000원이라는 숫자가 찍히는 순간, 눈물이 핑 돌았다.
```

**좋은 점:**
- 감각적 묘사 (잉크 냄새, 떨리는 목소리)
- 자연스러운 대화
- 숫자에 콤마 표기
- 감정선 (5년간의 노력 → 감격)
- Markdown 사용 안 함

## 🎯 톤 앤 매너

### 문체
- **평서문 중심:** "~했다", "~였다"
- **과거형:** 회고하듯이
- **3인칭 관찰자 + 1인칭 독백 혼용**

### 어휘
- 전문 용어는 최소화, 사용 시 즉시 설명
- 일상적이고 친근한 표현
- 은유와 비유 적절히 사용

### 감정 표현
- 과장하지 않되, 솔직하게
- "떨렸다", "두근거렸다", "한숨을 쉬었다" 등 구체적 표현
- 독백을 통한 내면 묘사
